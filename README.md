# 🚀 PersistenceAI Pro

> AI That Builds.

PersistenceAI Pro is a next-generation autonomous multi-agent AI operating system designed for development, automation, reasoning, and intelligent execution.

Built using Swift, Python, JavaScript, and modern AI infrastructure, PersistenceAI Pro combines powerful language models, memory systems, agent orchestration, and real-world tool execution into one unified platform.

- 💻 Autonomous Coding Agents

# ⚡ Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/PersistenceAI-Pro.git
cd PersistenceAI-Pro
```

---

## 2. Install Python Dependencies

```bash
pip install -r requirements.txt
```

---

## 3. Install Node Dependencies

```bash
npm install
```

---

## 4. Install Ollama

Download:

https://ollama.com

Run local AI model:

```bash
ollama run qwen2.5-coder
```

---

## 5. Start Backend

```bash
python main.py
```

---

## 6. Launch Frontend

```bash
npm run dev
```
